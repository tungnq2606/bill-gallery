## Project Overview

Bill Gallery — a personal mobile app for saving paper receipts and bank transfer screenshots as a visual gallery. Users scan/import bill images, detect amounts via OCR, split expenses, and track who still owes money. Local-first (SQLite), no account required.

**Tech Stack**: React Native 0.85.3 · Expo SDK 56 · TypeScript strict · Expo Router · Zustand · expo-sqlite · expo-camera · react-native-mlkit-ocr
